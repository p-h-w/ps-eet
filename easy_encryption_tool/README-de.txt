Skript zur einfachen Verschlüsselung zum sicheren und
datenschutzkonformen Upload von Konfigurationsdateien und Datenbanken
zum Support des SchulVerwaltungsProgramms. 


Notwendige Programmdateien

Für die Anwendung des Tools benötigen Sie das Archiv:

1. easy_encryption_tool.exe


Installation

Es ist keine Installation notwendig.


Anwendung

Gehen Sie folgendermaßen vor, um eine Datei zu verschlüsseln:

1. Rufen Sie das Programm "svp-encrypter.exe" mit einem Doppelklick
auf. Es öffnet sich nach einer kurzen Verzögerung ein Fenster.

2. Klicken Sie auf den Butten rechts neben dem Textfeld und wählen Sie
im Dateidialog die zu verschlüsselnde Datei aus. Beachten Sie, der Dateinamen
darf keine __ enthalten und auch kein Multidateiarchiv von 7z oder rar sein.

3. Wählen Sie ein Zielverzeichnis aus. Aus Gründen der Übersichtlichkeit 
empfiehlt es sich, ein neues, leeres Verzeichnis für die Zieldatei(en) 
anzulegen.

4. Klicken Sie auf "OK". 

5. Ist die Datei größer als 1GB, so wird sie in mehrere Teil zerlegt, die
wiederum verschlüsselt werden.

6. Im Konsolenfenster kann der Fortschritt des Vorgangs verfolgt werden. 
Ein Dialogfenster zeigt, dass die Verschlüsselung erfolgreich beendet wurde.

7. Laden Sie die ENC-Datei(en) auf dem vom Supportteam angelegten
Uploadlink hoch. Die Datei(en) enthält/enthalten neben dem ursprünglichen 
Dateinamen noch eine Zahlenfolge die eine Schlüsselzuordnung
ermöglicht. Bitte belassen Sie den Dateinamen so, wie er 
erstellt wurde.

ACHTUNG: Die verschlüsselte(n) ENC-Datei(en) kann von Ihnen NICHT MEHR
entschlüsselt werden! Löschen Sie also die unverschlüsselten
Originaldateien nur, wenn Sie sicher sind, dass diese nicht mehr
gebraucht werden.
