Version

0.1  Created decrypt tool from the encrypt tool and adapted the dialog.ps1 script. File selection dialog restricted to .enc files. Added password dialog.

0.1.1  Restored the original file extension from the filename.

0.1.2  Added the OpenSSL distribution to the tool path and call it from there to avoid requiring installations. Adjusted the script: added another environment variable.

0.1.3  Added selection of the target directory. Message box shows successful decryption at the end.

0.1.4  Added an option to choose an alternative key.

0.1.4.1  Changed method for selecting an alternative key. Added checkbox and text field.

0.1.4.2  a) Fixed bug in "Alternative key file" selection: if an alternative key was chosen it continued to be used even after unchecking the checkbox and required a restart to reset. Now, if the alternative key is not needed, unchecking the box will revert to the default key even if an alternative was previously selected.
b) Resolved security issues with the password string when passing it to OpenSSL. Password is no longer visible on the prompt.

0.1.4.3  Adjusted filenames for the key and certificate. Added an "old_key" directory.

0.1.4.4  Added option to show the password in plaintext. Added current key number display.

0.9  Switched to production certificates.

0.9.1  File stitcher for support of databases >1 GB. Multi-file archives are decrypted and automatically reassembled. Changed key directory for easier addition of new keys.

0.9.2  Used four digits for file suffixes in multi-file archives for compatibility with Linux split.

0.9.3  Added installation script keyimport/schluesselimport. Simplifies importing new keys and configuration files.

0.9.4  Implemented extensions/plugins architecture. Implemented a 7zip forward function to convert eet archives directly into 7z archives for transfer to the ISB.

0.9.5  Password generator for the 7zip tool. Functional update of the import tool. Only necessary variables are updated in the ini file.

0.9.6-3  Various bug fixes
